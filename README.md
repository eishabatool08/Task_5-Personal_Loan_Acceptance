# Personal Loan Acceptance Prediction 💳

## Project Objective
The goal of this project is to analyze customer data for a bank and build a classification model to predict which customers are most likely to accept a personal loan offer. This helps the bank target the right audience and increase their success rate.

## Key Features & Instructions
- **Data Cleaning:** Removed irrelevant features such as `ID` and `ZIP Code`.
- **Feature Engineering:** Added synthetic features like `Marital Status` for deeper analysis.
- **Exploratory Data Analysis (EDA):** Visualized the relationship between Income, Education, and Loan acceptance.
- **Model Building:** Implemented a classification model (Decision Tree / Logistic Regression).
- **Evaluation:** Evaluated model performance using Accuracy Score and Confusion Matrix.

## Dataset Highlights
The dataset contains information on 5,000 customers, including:
- **Demographics:** Age, Experience, Family size.
- **Financials:** Income, Average Credit Card spending, Education level.
- **Bank Relationship:** Fixed Deposit accounts, Security accounts, and Online banking usage.

## Tools & Libraries
- **Python** (Pandas, Numpy)
- **Visualization:** Seaborn, Matplotlib
- **Machine Learning:** Scikit-learn
